39   # A helper method to make SOAP calls for us. This method is mainly here to
 
40   # reduce code duplication: all SOAP calls expect a certain timeout and can
 
41   # tolerate certain exceptions, so we consolidate this code into this method.
 
42   # Here, the caller specifies the timeout for the SOAP call (or NO_TIMEOUT
 
43   # if an infinite timeout is required) as well as whether the call should
 
44   # be retried in the face of exceptions. Exceptions can occur if the machine
 
45   # is not yet running or is too busy to handle the request, so these exceptions
 
46   # are automatically retried regardless of the retry value. Typically
 
47   # callers set this to false to catch 'Connection Refused' exceptions or
 
48   # the like. Finally, the caller must provide a block of
 
49   # code that indicates the SOAP call to make: this is really all that differs
 
50   # between the calling methods. The result of the block is returned to the
 
51   # caller. 
52   def make_call(time, retry_on_except)
 
53     begin
 
54       Timeout::timeout(time) {
 
55         yield if block_given?
 
56       }
 
57     rescue Errno::ECONNREFUSED
 
58       if retry_on_except
 
59         retry
 
60       else
 
61         abort("Connection was refused. Is the AppController running?")
 
62       end
 
63     rescue OpenSSL::SSL::SSLError, NotImplementedError, Timeout::Error
 
64       retry
 
65     rescue Exception => except
 
66       if retry_on_except
 
67         retry
 
68       else
 
69         abort("We saw an unexpected error of the type #{except.class} with the following message:\n#{except}.")
 
70       end
 
71     end
 
72   end
